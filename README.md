# Meta Package Manager (mpm)

**Entirely NOT unctional at the moment.** See [https://github.com/eriknomitch/mpm](For the Ruby version).

## Usage

### OS Package Manager

```
mpm search <query>
mpm install <query>
mpm (uninstall|remove) <query>
mpm list [query]
mpm upgrade
mpm update <query>
mpm info <query>
mpm clean
```

### Package Managers

```
mpm /gem
mpm /brew
mpm /apt
mpm /cask
mpm /npm
```
